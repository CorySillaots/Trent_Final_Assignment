1.2 Operators

Math operators

Math operators perform mathematical calculations.

  • + addition of two numbers (also works on strings)
  • - subtraction (and unary operator which negates values) of two numbers
  • / division of two numbers
  • * multiplication of two numbers
  • % modulus (the number left over after division of 2 numbers)
  • ++ increment (count up by 1)
  • -- decrement (count down by 1)

Comparison operators

Comparison operators return a value of "true" or "false"

  • == "is equal to"
  • === "is exactly equal to, including type"
  • != "is not equal to"
  • > greater than
  • < less than
  • >= greater than or equal to
  • <= less than or equal to

Logic operators

Logical operators compare the left side value to the right side value and return "true" or "false". They can be strung together to get more complex results.

  • && (and) - must be true on both sides of the argument
  • || (or) - must be true on at least one side of the argument
  • ! (logical negative) - if there is a true anywhere in the argument, the result will be false, otherwise the result will be true.

Assignment operators

Assignment operators apply the right side value to the left side variable. In combination with some math operators, a calculation can be done first, with the result assigned to the left side value.

  • = assigns the value of the right side to the left side of the operator
  • += adds both numbers and assigns the result to the left side of the operator (also works with strings)
  • -= subtracts both numbers and assigns the result to the left side
  • *= multiplies and assigns
  • /= divides and assigns
  • %= modulus result and assign

String Operators

String operators work on string values similarily to how math operators work on numbers.

  • + concatenates two strings together
  • += concatenates two strings and assigns result to left side of operator

Specials - ternary

This special case is a short-form version of the If...Then...Else expression. It can be a bit difficult to read in coding, so it is best to use it for only simple items. It requires three pieces of information to work.

  • ? : - a simple if... then... else expression. The test is on the left side of the question mark, with the true (then) action placed between the" ?" and the ":" and the false (or else) action on the right of the ":"

General Syntax

The operator is placed between the two items, (variables, expressions, numbers, strings, etc.), that it is going to act upon. In general, the value on the right is assigned or compared to the value on the left of the operator.
