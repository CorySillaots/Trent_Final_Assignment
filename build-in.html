1.7 Built-in JS Objects

Javascript offers different built in objects to allow for more flexibility when writing code. The objects are Object, Array, String, Math and Date.

General Syntax

Object

let car = Honda

Array

let array =[a, b, c, d...]

String

let string = "This is a string"

Math (Math.floor() will round number down to nearest integer)

Math.floor(23.3365456);

Date

const d = new Date("2022-03-25");

Methods

Built in Objects are used for basic data processing and are uniquely used by different coders.

Examples

let answer = prompt("Please enter a number with a decimal spot");
let rounded = Math.floor(answer);
