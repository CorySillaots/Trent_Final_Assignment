2.3 HTML Objects

Objects are used in HTML to organize and implement different elements/features on a webpage.

General Syntax

An HTML tag name specific to the content of said tag wrapped in an opening and closing tag. See examples for more.

Methods

Using objects in HTML allows the functionality and compatibility with other coding languages.

Properties

An HTML object consists of an opening and closing tag wrapped around HTML code.

Examples

<section>Internal divs</section>
<div>HTML Code</div>
<p>HTML Code</p>
<button>Button Name</button>
