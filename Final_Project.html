1.1 Variables

A variable is a "container" for a piece of information or data. The data can be a wide variety of things like a number, letters, an object, a function, a path to a reference, or an expression, for example. A variable can be global: available to all parts of a script, or local: available to only specific parts of a script, like a specific function. A variable can be used to stream-line script writing, which will lessen errors, and also allow for greater ease-of-understanding when editing or updating a script at a later time.

General Syntax

Here are four examples of creating a variable and assigning a value to it. Variables must start with a letter and although they can contain numbers, they cannot start with one. Variables are also case-sensitive, and JavaScript has reserved certain words so they cannot be used as variable names

The key word "var" is the clue to declaring a variable.

var placeholder;
//this initializes the variable but does not assign a value to it (the auto-generated value is "undefined")

To assign a string or text value to a variable, you must use quotes around the value

var placeholder-text = "Some Text Here";
//this initializes AND assigns a string value to the variable

No quotes are needed to assign a numeric value.

var placeholder-number = 10;
// this variable has a number assigned to it

Variables can hold regular expressions as well, with the variable returning the result. Please remember that when a text value and a number value are added together, the result will be a text or string value.

var placeholder-both = (placeholder-text + placeholder-number);
//this variable has an expression as its value

The result of the last variable declaration would be "Some Text Here10".

Methods

You assign values to variables using the "=" operator, which means "assign the value on the right to the variable name on the left"

var GreetingString = "Hello ";

You can "add" variables together using the "+" operator.

var thanksForResponding = GreetingString + myName;

Properties

A variable may have properties depending on how it was created. If it was created within a function, it probably has properties. If it was created as a global variable, it probably does not.

Examples

//Script Sample for Variables 1.1
function runCodeVariables_onClick() {
//setting up the response text. Using "\n" to force a new line when needed
var GreetingString = "Hello ";
var ageString1 = "\nWOW, you are ";
var ageString2 = " years old!";
var contactString1 = "\nI will contact you by phone ";
var contactString2 = " or email ";

// getting information from the user for each item.
var myName = prompt("Please enter your name:", "");
var myAge = prompt("Please enter your age:", "");
var phone = prompt("Please enter your phone number:", "");
var email = prompt("Please enter your email address:", "");

//putting it all together into a single string for the alert message response
var thanksForResponding = GreetingString + myName + ageString1 + myAge +
ageString2 + contactString1 + phone + contactString2 + email + "\nThank you!";

//returning the assembled information in an alert message
alert(thanksForResponding);
}
