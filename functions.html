1.5 Functions

In Javascript, functions allow us to create a block of code that can be reused simply by calling the function and eliminates retyping code.

General Syntax

This is the general structure of a function:

function functionName(parameter 1, parameter 2){
block of code
}

Different types of values can be stored into an array such as numbers and strings and can be called based on an index number beginning at 0.

Methods

Functions are one of the key features of JavaScript and allow users to reuse blocks of code simply by calling the function. The blocks of code and be short, long, simple or complex.

Properties

A function is made up of the "function" keyword, the specific function name, parameters (which are optional) and the block of code the user wants executed when the function is called.

Examples

In this example, we will write a function that prompts the user for a number and then multiplies it by 10. The answer will then be displayed in an alert window.

function function_onClick(){
alert("Please enter two numbers to multiply together.")
let userInput = prompt("What is your first number?")
let userInputTwo = prompt("What is your second number?")
let answer = userInput * userInputTwo;
alert(`${userInput} x ${userInputTwo} = ${answer}`);
}
