1.3 Conditional Statements

A conditional statement is a three step process used in Javascript. Firstly, a condition is stated. There is then an expression to be executed if the condition is met and an expression to be execute if the statement is not met.

General Syntax

This is the general structure of an if/else statement:

var = (condition)
if (condition(s) to be met){ expression 1;
} else if (condition(s) to be met){
expression 2};

All if/else statements must have a at least a truthy and a falsy expression. You can make as many conditions as you'd like by using "else if".

This is the general structure for a switch statement.

switch(variable) {
case x:
expression 1
break;
case y:
expression 2
break;
default:
expression 3
}

No quotes are needed to assign a numeric value.

Methods

For an if/else statement, start by stating a condition to be met. In this example we will set three different outcomes depending on a score out of 10.

var score = x;
if (x => 8){
alert.window("Nice Job!");
} else if (x < 8 && x => 5){
alert.window("Not too shabby!");
} else (x < 5){
alert.window("Did you even study?");
}

For a switch statement, declare a variable that you will have multiple possible outcomes for. Similar to an if/else statement, state the condition to be met and the outcome once said condition is met. In this example we want to spell out a number that has been inputted by a user.

let userInput = prompt("Please enter a number between 1 and 5");
switch(userInput){
case 1:
userInput = "one";
break;
case 2:
userInput = "two";
break;
case 3:
userInput = "three";
break;
case 4:
userInput = "four";
break;
case 5:
userInput ="five";
break;
} alert.window(userInput + "is how you spell the number you entered.");

Properties

A variables properties can be modified by using conditional statements.

Examples

var answer = prompt("Rock, Paper or Scissors?");
if (answer = "rock"){
alert("You beat scissors but look out for paper!");
} else if (answer = "paper"){
alert("You beat rock but you're no match for scissors!");
} else if (answer = "scissors"){
alert("You cut paper but you'll never beat a rock!");
} else (
alert("No cheating! That's not in the game!")
)
}
