1.4 Loops

Loops are another feature Javascript uses to repeat tasks based on conditions and uses index numbers starting at 0.

General Syntax

A loop requires three parameters. Where to start, where to end and what increment to move through the index by. There is then a block of code to be executed each time the loop runs. This is the general structure of a Loop:

for (i = x; i < 5; i++){
document.getElementById("element_name").adjacentHTML = i; }

You can also create a loop within a loop. This is called a nest loop and looks like this:

for (i = x; i < 5; i++){
for (k = i; k < 5; k++){}
}

Methods

Imagine you need to display numbers 0 - 1000. It would take a long time to type out every number from 0 to 1000. Instead, we would use a for loop that knows to start at 0, increase by 1 until 1000 is reached and display each number each time the loop executes.

Properties

A for loop is made up of an index, starting point, end point, increment and code to be executed each time the loop runs.

Examples

In this example we will ask for a number. The for loop will then increase by 1 until we reach the number the user has input.

function loop_onClick(){
let endNum = prompt("What position in the index would you like your for loop to end at?")
let userInput = "";
for (let i = 0; i < endNum; i++) {
userInput += i + `< br/>`;
}
document.getElementById("loop").innerHTML = userInput;
}

It's important to remember that in this example we begin at 0. Therefore, stopping at the 3 position in the index will produce 0, 1, 2.
